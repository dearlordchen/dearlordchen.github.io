但是了解react的人应该都知道上面这种不是基于JSX的代码写起来太low了，而且html标签的优势也都消失殆尽了，需要把jsx模板转换的工具也集成进来，操作步骤如下：

  1. 在static/js 目录下新建目录jsx
  2. 下载安装nodejs

  3. 安装react-tools

  4. 设置jsx自动编译(记得关注文件路径，要先进入你本机代码所在目录)
